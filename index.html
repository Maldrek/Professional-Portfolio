Giphy Producer

With the Giphy producer page, a user is able to use a created button which returns the top 10 gifs for that search term calling from the Giphy API.

If the user instead has a different term to search for, users can enter their term in the input field and have a button dynamically created for them. They then can click that button to perform the API call to return the top 10 gifs related to the search term that they input.

Bamazon: The Amazon-esque Node Application

The Bamazon application uses Node.js, Javascript, and MySQL along with NPM package inquirer to mimic the processes of online purchasing. There is a customer and manager section that allows for different interactions with the database.

In the customer version of the app, the user is given the ID's and names of the items available for purchase, and then allows the user to select and item by ID and choose the amount they would like to purchase. If the amount is available, the user is given their total for the purchase and then are asked whether they would like to continue shopping or end the experience. If the amount ordered is over the stock quantity, then the user is given a message and are able to modify their order item or amount to order.

The manager functionality of the app allows the "manager" to see what items are available, check low inventory, add inventory if low, and also add new inventory.

Click here to view on Github!

Background Gradient Generator

This project built with HTML, CSS, and Javascript takes the current colors of the color picker input and creates a left-to-right two-tone background gradient with the CSS code to generate the same effect for any developer who uses it.

The user can change the color-gradient of the background simply by clicking on a color input and changing the color chosen to another. Because the color gradient is left-to-right, the pickers are ordered as such (the left color picker represents the starting gradient color, and the right represents the ending gradient color).
